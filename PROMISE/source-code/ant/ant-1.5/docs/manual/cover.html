Apache Ant 1.5 Manual

This is the manual for version 1.5 of Apache Ant. If your version of Ant (as verified with ant -version) is older or newer than this version then this is not the correct manual set. Please use the documentation appropriate to your current version. Also, if you are using a version older than the most recent release, we recommend an upgrade to fix bugs as well as provide new functionality
