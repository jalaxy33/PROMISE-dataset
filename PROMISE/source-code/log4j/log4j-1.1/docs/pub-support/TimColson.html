Dear Members of JSR47:
 
 Please consider adopting log4J instead of the proposed JSR47.

 It is my personal opinion that JSR47 is a redundant
effort to create a common, standardized Java service which
has already been built, refined, stress-tested, documented,
and used extensively by the Open Source community.

 Please support that community effort so that we may see
standardized logging implementations become commonplace
and empowering, rather than fractionated and confusing
to Java developers everywhere.

Thank you for your consideration!
