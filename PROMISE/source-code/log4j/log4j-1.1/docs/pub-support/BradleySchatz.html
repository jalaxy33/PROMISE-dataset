Dear Sir/Madam,

I urge you to reconsider the decision of including the JSR47 API as the
default logging API in the upcoming 1.4 release.

I consider the Log4j API to be the defacto standard in Java logging. It
is a mature and well undersood solution to the drivers of the JSR47 API.

By adopting JSR47 API as the primary logging API in Java 1.4 Sun will
contribute to bloating the Java platform, and weaken the Java community.
